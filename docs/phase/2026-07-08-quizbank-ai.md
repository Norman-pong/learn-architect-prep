### 2.2 任务详情

#### T001: 题库拉取与去重（远程 URL + SHA-256）

**任务描述**：实现从远程源拉取题库 JSON，使用 SHA-256 对题目内容进行去重，并导入本地题库。

**验收标准**：
- [ ] 可配置远程题库 URL 并拉取
- [ ] 使用 SHA-256 对题目内容去重
- [ ] 去重后题目可正确写入本地题库
- [ ] 提供拉取日志和去重统计

#### T002: 案例题/论文题库与 AI 评分标准

**任务描述**：调研并整理案例题和论文题库，明确 AI 评分标准（案例题 4 维度、论文 5 维度）。

**验收标准**：
- [ ] 案例题题库可拉取并按科目组织
- [ ] 论文题库含主题、范文、评分标准
- [ ] AI 评分 prompt 与评分维度一致

#### T003: AI 智能选题（从题库筛选，不生成）

**任务描述**：根据薄弱点从现有题库中筛选相关题目，不向 LLM 请求生成新题。

**验收标准**：
- [ ] 薄弱点可触发智能选题
- [ ] 推荐题目全部来自现有题库
- [ ] 选题结果与用户薄弱知识点相关

#### T004: AI 论文评分（5 维度 + streamText）

**任务描述**：使用 Vercel AI SDK 的 streamText 实现论文 5 维度评分（切题 30%/应用深度 20%/实践性 20%/表达 15%/综合 15%）。

**验收标准**：
- [ ] 5 维度评分结构正确
- [ ] 返回逐段点评和改进建议
- [ ] 评分响应以 stream 形式逐步呈现
- [ ] AI 响应 < 30s

#### T005: 题目仓库级编辑（报告错误）

**任务描述**：应用内提供「报告错误」入口，将题目问题提交到题库 Git 仓库进行协作修正。

**验收标准**：
- [ ] 用户可针对题目提交错误报告
- [ ] 报告信息包含题目 ID、错误类型、描述
- [ ] 错误报告可汇总到题库仓库 issue 或 PR
