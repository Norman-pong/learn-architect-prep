### 2.2 任务详情

#### T001: 知识点体系管理（20 章 + 5 分制标注）

**任务描述**：按官方教材 20 章结构组织知识点，每章知识点按考试重点用 1-5 分制标注，并渲染为树形导航。

**验收标准**：
- [ ] 20 章知识点 Markdown 文件已组织完毕
- [ ] 5 分制标注准确且前端可正确渲染颜色
- [ ] 树形导航支持展开/收起并跳转详情

#### T002: 间隔重复调度 SM-2

**任务描述**：用 SM-2 算法变体管理每日复习队列，支持 0-5 分评分并更新 ease 和 interval。

**验收标准**：
- [ ] 每日复习队列按 due_date 升序生成
- [ ] 评分 0-2 重置 interval 为 1，ease 衰减
- [ ] 评分 3/4/5 按 SM-2 规则更新 interval 和 ease
- [ ] 连续 7 天队列调度正确

#### T003: 学习仪表盘

**任务描述**：首页聚合复习队列、学习天数、薄弱点预警、练习成绩等数据。

**验收标准**：
- [ ] 仪表盘显示今日复习数量
- [ ] 显示连续学习天数
- [ ] 显示薄弱点预警
- [ ] 显示近期练习成绩趋势

#### T004: 薄弱点识别

**任务描述**：基于正确率和模考得分率识别薄弱知识点，正确率 < 60% 或模考得分率 < 50% 标记为薄弱。

**验收标准**：
- [ ] 薄弱点按知识点/章节聚合
- [ ] 正确率计算准确
- [ ] 薄弱点预警可在仪表盘查看

#### T005: 选择题练习模式（章节/随机/错题）

**任务描述**：支持三种练习模式：按章节练习、随机练习、错题练习。

**验收标准**：
- [ ] 章节模式按选择章节出题
- [ ] 随机模式从题库随机抽取
- [ ] 错题模式从用户错题本中出题
- [ ] 答题后实时反馈正误并解析

#### T006: 题库管理 + 真题导入

**任务描述**：建立静态题库 JSON Schema，支持历年真题导入（2009-2025），题目与知识点关联。

**验收标准**：
- [ ] 题库 JSON 结构统一
- [ ] 题目可按年份、科目、章节筛选
- [ ] 真题导入脚本可复用

#### T007: 错题本

**任务描述**：练习中答错的题目自动进入错题本，支持标记已掌握、按章节筛选、重新练习。

**验收标准**：
- [ ] 错题自动收录
- [ ] 可标记已掌握并从错题本移除
- [ ] 支持按章节筛选

#### T008: 练习统计分析

**任务描述**：统计用户练习数据，生成正确率、薄弱点、学习趋势等分析结果。

**验收标准**：
- [ ] 正确率按知识点和章节聚合准确
- [ ] 统计分析支持时间维度筛选
- [ ] 数据驱动薄弱点识别合理
