## 1. 阶段目标

### 1.1 阶段定位

本阶段在核心功能基础上实现 P1 级增强功能，包括学习进度可视化、AI 答疑、全文搜索、复习推荐、学习提醒、成绩趋势、写作技巧、AI 成本展示以及数据导入导出，并补齐重点标注等 P2 功能。

### 1.2 阶段目标

- 完成学习增强功能（进度追踪、AI 答疑、搜索、推荐）。
- 完成辅助功能（提醒、成绩趋势、写作技巧、AI 成本、导入导出）。
- 完成 P2 级重点标注功能。

### 1.3 完成标准

- [ ] P1 功能按 PRD 要求全部实现并通过验收。
- [ ] 搜索、推荐、提醒等数据驱动功能结果合理且稳定。
- [ ] AI 平均响应时间符合 PRD 性能指标（< 30s）。

### 2.2 任务详情

#### T001：学习进度追踪（热力图+学习日历）

- **任务描述**：根据用户学习记录生成热力图和学习日历，展示每日学习时长、做题量与复习进度。
- **验收标准**：
  - [ ] 热力图按日期展示学习强度。
  - [ ] 学习日历可查看每日详细记录。
- **涉及文档**：PRD §3.1 FR-LR-03
- **备注**：无

#### T002：AI 知识点答疑（对话式+引用章节）

- **任务描述**：提供对话式 AI 答疑，回答需引用相关知识点章节或教材来源。
- **验收标准**：
  - [ ] 多轮对话上下文保持连贯。
  - [ ] 回答中引用章节信息准确。
- **涉及文档**：PRD §3.1 FR-LR-04
- **备注**：Prompt 设计需多次调优。

#### T003：知识点搜索（全文搜索）

- **任务描述**：支持对知识点、笔记、错题、范文等内容的全文搜索。
- **验收标准**：
  - [ ] 搜索结果相关性可接受。
  - [ ] 支持关键词高亮。
- **涉及文档**：PRD §3.1 FR-LR-06
- **备注**：无

#### T004：复习推荐（薄弱程度×考试权重）

- **任务描述**：基于用户错题、薄弱知识点与考试权重生成复习推荐列表。
- **验收标准**：
  - [ ] 推荐结果综合考虑薄弱度与权重。
  - [ ] 推荐可触发复习任务或习题练习。
- **涉及文档**：PRD §3.1 FR-PF-02
- **备注**：无

#### T005：学习提醒（邮件提醒复习队列）

- **任务描述**：根据复习队列向用户发送邮件提醒，支持提醒规则配置。
- **验收标准**：
  - [ ] 邮件提醒可成功发送。
  - [ ] 提醒内容包含当日复习任务摘要。
- **涉及文档**：PRD §3.1 FR-PF-04
- **备注**：定时任务需可靠运行。

#### T006：成绩记录与趋势（分数曲线）

- **任务描述**：记录模拟考成绩并绘制分数趋势曲线，支持按科目筛选。
- **验收标准**：
  - [ ] 历史成绩可持久化查询。
  - [ ] 趋势曲线展示准确。
- **涉及文档**：PRD §3.1 FR-EX-04
- **备注**：无

#### T007：写作技巧指导（历年题目+选题策略）

- **任务描述**：提供历年论文题目汇总与选题策略指导，帮助用户备考论文。
- **验收标准**：
  - [ ] 历年题目数据完整。
  - [ ] 选题策略建议可用。
- **涉及文档**：PRD §3.1 FR-WR-04
- **备注**：无

#### T008：AI 成本展示（token 消耗+费用）

- **任务描述**：在后台或用户可见位置展示 AI 调用的 token 消耗与预估费用。
- **验收标准**：
  - [ ] token 消耗统计准确。
  - [ ] 费用估算符合模型单价配置。
- **涉及文档**：PRD §3.1 FR-SY-04
- **备注**：无

#### T009：数据导入导出（JSON 备份/恢复）

- **任务描述**：支持用户学习数据、错题、笔记、论文草稿的 JSON 格式导出与导入恢复。
- **验收标准**：
  - [ ] 导出数据完整、可解析。
  - [ ] 导入后可恢复对应数据。
- **涉及文档**：PRD §3.1 FR-SY-03
- **备注**：无

#### T010：知识点重点标注（高亮/笔记/疑问）

- **任务描述**：允许用户在知识点内容上做高亮、添加笔记或标记疑问。
- **验收标准**：
  - [ ] 高亮、笔记、疑问标记可持久化。
  - [ ] 标记可在复习时单独查看。
- **涉及文档**：PRD §3.1 FR-LR-05
- **备注**：P2 功能，核心路径完成后实现。
